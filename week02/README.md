# Week 02 - Advanced JavaScript

This week dives deeper into modern JavaScript (ES6+) features, object-oriented programming with classes, and advanced array manipulations.

## Topics Covered
- **ES6+ Features**: Using Spread (`...`) and Rest operators.
- **Classes**: Introduction to Class-based programming in JavaScript.
- **Advanced Array Operations**: Map, Filter, Reduce, and other functional programming patterns.
- **Memory & Reference**: Understanding Shallow Copy vs. Deep Copy.
- **Asynchronous JavaScript**: Working with Timers (`setTimeout`, `setInterval`).

## File Descriptions
- `Class_demo.js`: Demonstrates class definitions and inheritance.
- `Array_operations.js`: Basic and intermediate array methods.
- `Array_op_advanced.js`: Advanced usage of `map`, `filter`, and `reduce`.
- `SpreadOp_nd_RestPara.js`: Practical examples of spread and rest syntax.
- `Shallow_nd_Deepcpy.js`: Comparison between shallow and deep copying of objects and arrays.
- `Timer_fun.js`: Examples of using JavaScript timers for asynchronous execution.
