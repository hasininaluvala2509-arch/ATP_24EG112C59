# Week 01 - JavaScript Fundamentals

This week focuses on the foundational concepts of JavaScript, including basic syntax, data types, control flow, and functions.

## Topics Covered
- **Operators**: Arithmetic, relational, and logical operators.
- **Control Statements**: `if-else`, loops (`for`, `while`), and switch cases.
- **Functions**: Defining and invoking functions, working with arguments.
- **Arrays**: Basic array operations, finding indices, and calculating sums.
- **Objects**: Basic object creation and property management.

## File Descriptions
- `operators.js`: Demonstrates various JavaScript operators.
- `control_statements.js`: Examples of conditional and iterative logic.
- `greatest_of_2.js` / `greatest_of_3.js`: Logic to find the largest number.
- `fun_3args.js`: Working with function parameters.
- `array_sum.js`: Logic to calculate the sum of array elements.
- `find_smallest.js`: Finding the minimum value in an array.
- `EmployeeObj.js`: Introduction to JavaScript objects using an employee example.
