# Week 6 - MERN Mini Project

This directory contains a mini application built using the MERN stack (MongoDB, Express, React, Node.js).

## Project Overview
- **mern-mini-app**: A consolidated project featuring both a backend and a frontend for managing employee data.

## Folder Structure
- `emp_backend/`: Contains the Express.js server, database models, and API endpoints.
- `emp_frontend/`: Contains the React.js application with components and styling.

## Learning Objectives
- Understanding the end-to-end flow of data from a React frontend to a MongoDB database.
- Implementing CRUD (Create, Read, Update, Delete) operations in a full-stack context.
