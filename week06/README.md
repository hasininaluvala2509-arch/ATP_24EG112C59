# Week 06 - State Management & Full-Stack Integration

This week explores advanced state management in React using the Context API and continues building full-stack applications with Employee management projects.

## Topics Covered
- **React Context API**: Managing global state without prop drilling.
- **Full-Stack Development**: Integrating React frontends with Express backends.
- **Employee Management System**: A practical implementation of a MERN application.
- **Modular Project Structure**: Organizing backend and frontend into separate directories.

## File Descriptions
- `CounterContextProvider/`: Examples of implementing global state using `Context.Provider`.
- `emp_backend/`: The Node.js and Express server for the Employee management system.
- `emp_frontend/`: The React application for the Employee management system.
- `react-app-1/`: Additional React exercises and component practice.
