# Week 04 - Web Styling & Layouts

This week focuses on creating responsive web pages using HTML and CSS, with a focus on layouts and styling assignments.

## Topics Covered
- **Semantic HTML**: Structuring web content for accessibility and SEO.
- **CSS Layouts**: Working with Flexbox, Grid, and traditional box models.
- **Responsive Design**: Creating pages that work across different screen sizes.
- **External CSS**: Organizing styles in separate `.css` files.

## File Descriptions
- `Cities/`: A collection of HTML and CSS files for different city pages (Bangalore, Chennai, Hyderabad).
- `ass1.html`, `ass2.html`, `ass3.html`: Assignment files demonstrating various layout techniques and CSS properties.
- `Cities/cities.html`: A central hub page for the city-themed styling project.
