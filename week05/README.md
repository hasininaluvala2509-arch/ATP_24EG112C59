# Week 05 - Introduction to React & Tailwind CSS

This week introduces modern frontend development using React and utility-first styling with Tailwind CSS.

## Topics Covered
- **React Components**: Building reusable UI components.
- **Hooks**: Using `useState` and `useEffect` for state and side effects.
- **Tailwind CSS**: Styling applications using utility classes.
- **API Consumption**: Fetching data from backend APIs within React.

## File Descriptions
- `reactapp/`: A complete React application project.
    - `App.jsx`: The main application component with various demos.
    - `APIDemo.jsx`: Demonstrates fetching and displaying data from an API.
    - `Counter.jsx`: A simple state management example.
- `tailwind/`: Exercises focused on styling with Tailwind CSS.
- `QuickBlog/`: A mini blog project starting the transition to the capstone project.
- `table/`: Exercises for creating and styling tables in HTML/React.
